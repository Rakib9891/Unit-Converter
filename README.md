# Unit Converter

A simple Unit Converter built with HTML, CSS, and JavaScript.

## Features

- Convert Length units
- Convert Weight units
- Convert Temperature units
- Responsive and easy-to-use interface
- Instant conversion results
- Reset functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Supported Units

### Length
- Millimeter (mm)
- Centimeter (cm)
- Meter (m)
- Kilometer (km)
- Inch (in)
- Foot (ft)
- Yard (yd)
- Mile (mi)

### Weight
- Milligram (mg)
- Gram (g)
- Kilogram (kg)
- Ton (t)
- Ounce (oz)
- Pound (lb)

### Temperature
- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
